## How to install:

For developers:
```sh
curl -fsSL https://devexp.pro/install.sh | sh
```

By tag:
```sh
deno -A -r https://devexp.pro/tag.ts [tag] [alias_name]
```

By branch:
```sh
deno -A -r https://devexp.pro/branch.ts [branch_name] [alias_name]
```
